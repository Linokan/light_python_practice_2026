Консольный индексатор папок
Описание проекта

Проект представляет собой консольное приложение на Python для индексирования файлов в папках и хранения информации о них в базе данных SQLite.

Программа позволяет:

рекурсивно сканировать папки;
сохранять информацию о файлах в SQLite;
фильтровать файлы по расширению;
вычислять SHA-256 хэши файлов;
находить дубликаты файлов;
сравнивать основную папку с резервной копией и находить различия.
Используемые технологии
Python 3
SQLite (sqlite3)
Стандартная библиотека Python
Структура проекта
src/
├── main.py          # точка входа
├── scanner.py       # сканирование файлов
├── db.py            # работа с SQLite
├── duplicates.py    # вычисление хэшей и поиск дубликатов
├── backup.py        # сравнение папок
├── student_meta.md  # информация о студенте

data/
└── app.db           # база данных SQLite

journal.md           # журнал выполнения практики
README.md            # описание проекта
Возможности программы
Сканирование папки

Программа рекурсивно обходит указанную папку и собирает:

относительный путь файла;
размер файла;
время последнего изменения;
расширение файла;
SHA-256 хэш содержимого.
Фильтрация по расширению

Поддерживается фильтрация файлов по расширению.

Пример:

python src/main.py scan . --ext .py
Поиск дубликатов

Дубликаты определяются по одинаковому SHA-256 хэшу содержимого файла.

Результат выводится в консоль.

Сравнение резервной копии

Программа умеет сравнивать две папки и определять:

отсутствующие файлы;
измененные файлы;
лишние файлы.

Пример:

python src/main.py compare source backup
Создание базы данных

При первом запуске автоматически создается база данных:

data/app.db

В базе хранится индекс файлов:

путь;
размер;
время изменения;
расширение;
SHA-256 хэш.
Примеры запуска

Сканирование всех файлов:

python src/main.py scan .

Сканирование только Python-файлов:

python src/main.py scan . --ext .py

Сравнение основной папки и резервной копии:

python src/main.py compare source backup
